4.在IntelliJ IDEA中配置maven

在idea中点击file>settings>搜索 maven

注意还要在others settings> defaults settings中进行全局设置，为了防止切换工程时用不了

5.idea中创建maven的web工程

打开 idea，选择创建一个新工程

选择 idea提供好的 maven 的web 工程模板

点击 Next填写项目信息

点击 Next，添加的配置为 archetypeCatalog=internal

点击 Next选择项目所在目录:

点击 Finish后开始创建工程，耐心等待，直到出现如下界面。

可能遇到的问题:

Maven execution terminated abnormally (exit code 1)


Maven->Runner: 添加-Dmaven.multiModuleProjectDirectory=$M2_HOME ，记得选择JDK

手动添加 src/main/java 目录，如下图右键 main 文件夹>New>Directory

创建一个新的文件夹命名为 java

点击 OK 后，在新的文件夹 java 上右键>Make Directory as>Sources Root


也可以右键项目-选择Open Module Settings打开项目配置页面更改

添加 tomcat7 插件

<plugins><plugin><groupId>org.apache.tomcat.mavengroupId><artifactId>tomcat7-maven-pluginartifactId><version>2.1version><configuration><port>80port><path>/path>configuration>plugin>plugins>

手动输入 tomcat7:run 命令运行项目

补充：

Maven下载jar包速度过慢的解决

https://developer.aliyun.com/mvn/guide

在maven的settings.xml文件里的mirrors节点，添加如下子节点：

<mirror><id>nexus-aliyunid><mirrorOf>centralmirrorOf><name>Nexus aliyunname><url>http://maven.aliyun.com/nexus/content/groups/publicurl>mirror>
